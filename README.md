# gLog

Your silver bullet logging solution.

## Features

gLog supports the following features:

- log-levels
- smart-colors
- customization
- caller-insertion
- mutiple outputs
- log rotation
- panic-handling
- conditionals:
	- output per logging level
	- caller from a certain level
- sane defaults
